# Judges Card

A clean, professional scoring card application for judges. Perfect for dog shows, competitions, and trials.

## Features

- ✅ Trial information tracking (Trial #, Class, Handler, Judge)
- ✅ Scoring system with faults tracking
- ✅ Multiple run management
- ✅ Elimination marking
- ✅ Strength/weakness notes
- ✅ Export functionality
- ✅ Mobile-responsive design
- ✅ Professional printing support

## Live Demo

Open `index.html` in any modern web browser to start using it immediately.

## Usage

1. Enter trial information (number, class, handler, judge)
2. Add runs with handler names
3. Enter scores and faults for each run
4. Mark eliminations if needed
5. Add notes on strengths/weaknesses
6. Export or print your card
